# Salary-Prediction-System
Global Developer Salary Estimator (Regression ML System)

~ Built end-to-end regression system to predict software developer salaries using Stack Overflow 2025 survey dataset.

~ Performed data validation, schema consistency checks, and preprocessing on large-scale survey data with missing values and duplicates.

~ Engineered feature transformation pipelines using ColumnTransformer with imputation, scaling, and one-hot encoding to prevent data leakage.

~ Trained and compared Linear Regression, SVM, Decision Tree, Random Forest, and XGBoost models using cross-validation.

~ Selected XGBoost as final model achieving MAE of 42,865.

~ Deployed production-ready Streamlit application for real-time global salary estimation.
